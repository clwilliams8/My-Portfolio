Tutor Chest

About Tutor Chest

Tutor Chest is an EdTech platform designed to create an accessible online environment that enhances learning outcomes for students, parents, and educators. The platform connects families with qualified educators, offers professional development resources, and fosters a collaborative learning community. Learn more on their about page.

About The Project

Mission: Rescue and modernize a struggling EdTech platform following multiple unsuccessful development engagements, delivering seamless cross-platform integration and robust payment infrastructure.

Challenge: The client had engaged a web development agency and two freelancers, all of whom failed to deliver satisfactory results. The existing Laravel application served two user types—educators offering tutoring packages and parents searching for services—but the client faced critical issues: inability to make content updates independently (with the previous agency quoting prohibitive fees for simple changes), and failed attempts to integrate WordPress for content management while maintaining a unified domain experience.

Solution: I engineered a custom WordPress plugin exposing an API endpoint for retrieving WordPress content within the Laravel application, enabling unified domain presentation at "tutorchest.com" without subdomain complexity. I developed custom Laravel middleware to dynamically fetch and render WordPress pages on demand, creating a seamless user experience across both platforms. Additionally, I implemented Stripe payment processing via Laravel Cashier for subscription management, delivered UI/UX improvements across the Laravel application, and optimized the WordPress theme for mobile responsiveness.

Technologies Used: Laravel (Backend Framework), WordPress REST API (Content Management Integration), Laravel Cashier + Stripe (Payment Processing), Laravel Forge (Server Provisioning), Bitbucket (Version Control), Tailwind CSS (Frontend Styling).
